# 🚀 keysstudio

> lightweight command runner for repetitive tasks

## ✨ Features

- Fast compilation (sub-second builds)
- Hot reload included
- Plugin system
- Zero config start

## 📦 Installation

```bash
npm install -g keysstudio
keysstudio init my-project
```
